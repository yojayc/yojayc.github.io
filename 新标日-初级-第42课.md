## 课文

テレビをつけた<span style="color:red">まま</span>、出（で）かけてしまいました
开着电视就出门了

目覚（めざ）ましをかけておいた<span style="color:red">のに</span>、今朝（けさ）は起（お）きられませんでした
尽管上了闹钟，今天早上还是没起来

会議（かいぎ）は5時（ごじ）までですから、もうすぐ終（お）わる<span style="color:red">はず</span>です
会议5点钟结束，应该快完了

張（ちょう）さんは入院中（にゅういんちゅう）ですから、旅行（りょこう）に行（い）く<span style="color:red">はずがありません</span>
小张在住院，不可能去旅行

## 语法

### 小句1（动词<span style="color:red">た形</span>/<span style="color:red">ない形</span>）まま，小句2 / 名词 + <span style="color:red">の</span> + まま，小句

表示**将理应改变的状态<span style="color:red">保持着</span>去进行另外的动作**（多数是负面的）。まま前不能用敬体形（食べる物、行く場所　食べます物X　行きます場所X）

テレビを　つけたまま、出かけて　しまいました

李（り）さんはコートを　着（き）たまま、ベッドで寝（ね）て　しまいました
小李穿着外套就躺在床上睡了

馬（ば）さんは行（い）き先（さき）を　告（つ）げないまま、家（いえ）を出（で）て　しまいました
小马没有说去哪里就离开了家

私たちは　空腹（くうふく）のまま、出発（しゅっぱつ）しました
我们空着肚子就出发了

小李穿着拖鞋就去面试了
李さんはスリッパを　履（は）いたまま、面接（めんせつ）にいってしまいました

我穿着拖鞋就去跑步了
私はスリッパを　履いたまま、ジョギング　に行きました

### 小句1（<span style="color:red">简体形</span>）のに，小句2

除了表示[**用途**](https://yojayc.github.io/2026/03/21/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC36%E8%AF%BE/#%E5%8A%A8%E8%AF%8D%EF%BC%88%E5%9F%BA%E6%9C%AC%E5%BD%A2%EF%BC%89%E3%81%AE-%E5%90%8D%E8%AF%8D-%E3%81%AB)，还表示**理应发生但是没有发生**，翻译成“**明明……**”。のに前一般不用敬体形

**二类形容词/名词 + <span style="color:red">な</span> + のに**

> のに 前**不能接现在时基本形**
>
> のに 表示**转折遗憾**，前面必须是已经发生的事，用**过去式た形**

目覚ましをかけておいたのに、今朝は起きられませんでした

今日（きょう）はこんなに寒（さむ）いのに、森（もり）さんは寒（さむ）くないと言（い）いました
尽管今天很冷，森先生却说不冷

この家（いえ）は駅（えき）から近（ちか）くて便利（べんり）なのに、買（か）う人（ひと）がいません
这个房子离车站近而方便，却没有人购买

田中（たなか）さんは作家（さっか）なのに、読書（どくしょ）が嫌（きら）いです
田中先生尽管是作家却讨厌读书

森さんは昨日（きのう）病気（びょうき）だったのに、出勤（しゅっきん）しました
森先生昨天生着病却去上班了

> [**小句+が**](https://yojayc.github.io/2026/01/29/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC16%E8%AF%BE/#%E5%B0%8F%E5%8F%A5-%E3%81%8C-%E5%B0%8F%E5%8F%A5-%E8%BD%AC%E6%8A%98) 情绪**轻微**的转折语气。前后句意思相反时使用。小句1可以是**です、ます**。小句2可以是**祈使句请求句**。[**けど**](https://yojayc.github.io/2026/02/09/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC22%E8%AF%BE/#%E3%81%91%E3%81%A9-%E8%BD%AC%E6%8A%98-%E9%93%BA%E5%9E%AB) 是 が 的口语形式
>
> **小句+のに** 情绪**强烈**的转折语气。违背预想结果（**多数带负面情绪**）时使用。小句1不能是**です、ます**。小句2<span style="color:red"><strong>不能是祈使句请求句</strong></span>

この料理は美味しいですが、ちょっと高いです
この料理はまずいのに、すごく高いです

寒いですが、換気のために窓を　開けてください
寒いのに、換気のために窓を　開けてください　X

### 小句（<span style="color:red">简体形</span>）はずです

强调**主体本人的<span style="color:red">推测</span>**，主体对所推测的内容非常有把握

**二类形容词 + <span style="color:red">な</span> + はずです**

**名词 + <span style="color:red">の</span> + はずです**

会議は5時までですから、もうすぐ　終わるはずです

この映画（えいが）は人気（にんき）がありますから、観客（かんきゃく）は　多（おお）いはずです
这部电影很受欢迎，看得人应该很多

キムさんはこの大学（だいがく）の　学生（がくせい）のはずです
金女士应该是这所大学的学生

小林应该很喜欢漂亮的女生
小林君（こばやしくん）は　きれいな女性/可愛い女性/美人（びじん）**が**　好き**な**はずです

小林应该在海边看美女
小林君は　海で　美女（びじょ）たちを　見ている　はずです

> はずです 既可用于说话人自身经历过但**记忆已经十分不准确**，也可用于实际情况与自己的**考虑不一致**

たしか、書類（しょるい）は机（つくえ）の上（うえ）に置（お）いたはずです
我记得文件应该是放在桌子上了

清水（しみず）さんはまだ来（き）ていませんが、今日（きょう）は来（く）るはずです
清水先生还没有来，不过今天应该会来

### 小句（<span style="color:red">简体形</span>）はず<span style="color:red">が</span>ありません

**～はずがありません** 是 **～はずです** 的<span style="color:red"><strong>否定</strong></span>形式，也可以说成 <strong>～はず<span style="color:red">は</span>ないです</strong>

**二类形容词 + <span style="color:red">な</span> + はずがありません**

**名词 + <span style="color:red">の</span> + はずがありません**

張さんは入院中ですから、旅行に　行くはずがありません

7月（しちがつ）のオーストラリアは冬（ふゆ）ですから、暑（あつ）いはずがありません
7月的澳大利亚正好是冬季，不可能热

太田（おおた）さん、遅（おそ）いですね
太田先生真够晚的
ーー　ええ。でも、太田（おおた）さんは責任者（せきにんしゃ）ですから、遅（おく）れるはずはないんですが
—— 是啊。太田先生是负责人，不应该迟到呀

森さんはスキーがあんなに上手（じょうず）ですから、初心者（しょしんしゃ）のはずがありません
森先生的滑雪技术那么好，不可能是初学者

芋泥桑日语说的那么好，不可能是初学者
ユニちゃんの　日本語は　あんなに上手ですから、初心者のはずがありません

## 表达及词语讲解

### 目覚（めざ）ましをかけます

目覚まし 是 目覚まし時計（どけい）的省略说法

设定时间一般说 目覚ましをかけます、目覚ましをセットします、アラームをセットします

### 〜ですもの

表示原因、理由。具有一种申明自己行为是正当的语感。多为女性使用

### せっかく〜のに

[**せっかく**](https://yojayc.github.io/2026/03/25/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC39%E8%AF%BE/#%E3%81%9B%E3%81%A3%E3%81%8B%E3%81%8F)**～のに** 表示对自己或对方的付出没有得到相应的回报或预期效果而表示惋惜、遗憾的心情

### 早速（さっそく）

表示迅速进行某种行为的情况

与 **すぐ** 相比，早速 表示说话人或听话人积极主动开始行动，不能用于客观描写

在书信中 **早速** 不能用 **すぐ** 替代
