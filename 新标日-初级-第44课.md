### 小句（简体形）ようです [推测]

**名词 + <span style="color:red">の</span> + ようです**

**一类形容词 + ようです**

**二类形容词 + <span style="color:red">な</span> + ようです**

表示**根据说话人感知（视觉、听觉、触觉、味觉、嗅觉等）的某种情况进行推测**

玄関のところに　誰かいるよう　です

エンジンが　故障（こしょう）したよう　です
好像是发动机出故障了

店（みせ）の前（まえ）には行列（ぎょうれつ）ができています。あの店の料理は　おいしいよう　です
那个饭店前面排起了长队，看来菜的味道不错

値段（ねだん）は他（ほか）の店（みせ）の倍（ばい）です。ここの料理（りょうり）はかなり　豪華（ごうか）なよう　です
价钱是别的饭店的两倍，这儿的菜看起来很豪华

遠（とお）くから何（なに）か飛（と）んできます。　飛行機（ひこうき）のよう　ですね
什么东西从远处飞过来了，好像是飞机
ーー　そのようですね
—— 好像是

### 小句（简体形）みたいです [推测]（口语）

**名词 / 一类形容词 / 二类形容词 + みたいです**

**将二类形容词 / 名词简体形的 <span style="color:red">だ</span> 换成 <span style="color:red">みたいです</span>**

小野さんは森さんをよく見（み）ていますね。小野さんは森さんが　好きみたいです
小野女士老是看森先生。她好像喜欢森先生

雨（あめ）の音（おと）が聞（き）こえません。雨（あめ）が　やんだみたいです
听不见雨声了。好像雨停了

外（そと）の人（ひと）はみんなコートを着（き）ています。外（そと）は　寒（さむ）いみたいです
行人都穿着大衣。外面好像很冷

雲（くも）が広（ひろ）がっています。明日（あした）は　雨みたいですよ
云层向四周蔓延。明天好像要下雨

### 小句（简体形）らしいです [传闻] 

**名词 / 一类形容词 / 二类形容词 + らしいです**

**将二类形容词 / 名词简体形的 <span style="color:red">だ</span> 换成 <span style="color:red">らしいです</span>**

今度の社員旅行は韓国へ　行くらしいです

李（り）さんは頭（あたま）が　痛（いた）いらしいです
小李好像头疼

林（はやし）さんはお酒（さけ）が　好（す）きらしいですよ
林先生好像喜欢喝酒

電気（でんき）が消（き）えています。李さんは　出（で）かけたらしいです
电灯关着。小李大概是出去了

パトカーのサイレンが聞（き）こえます。近所（きんじょ）で事故（じこ）が　あったらしいです
狗车叫呢。这附近好像出什么事儿了

### 动词 / 一类形 / 二类形 過（す）ぎます

表示某种动作或事物的性质**超过了正常的量或程度**

**动词连用形 + 過ぎます**

**一类形容词把词尾 “<span style="color:red">い</span>” 换成 “過ぎます”**

**二类形容词直接加過ぎます**

昼ご飯を食べ過ぎました

このスープは熱（あつ）過ぎませんか
你不觉得这个汤太烫吗？

この説明（せつめい）は複雑（ふくざつ）過ぎます
这说明太复杂

这道题太难了
この問題は難し過ぎます

阿飞老师太亲切了
トビ先生は親切（しんせつ）過ぎます

### 一类形容词 + さ

将部分一类形容词词尾 **い** 换成 **さ** 构成一种表示某种状态的名词

この 暑（あつ）さ はいつまで続（つづ）くんでしょう
这种炎热的天气要持续到什么时候呢？

富士山（ふじさん）の 美（うつく）しさ が心（こころ）に残（のこ）っています
富士山的美留在了我的心底

仕事（しごと）の おもしろさ がやっと分（わ）かりました
终于体会到了工作的趣味

## 表达及词语讲解

### おかげさまで

用于对对方的关心、惦念

### けっこう

表示受欢迎的程度比自己预想的要高

けっこう長い、けっこう高い

### 何（なん）と言（い）っても

表示**从多种因素中着重选择一种**

### デザインがしゃれていて

表示产品的设计新颖

日语动词表示某种性质时，需用 **～ています** 的形式

### 受（う）ける

表示受到别人赞赏或欢迎

### 控（ひか）えめ

表示分量少和做事有节制
