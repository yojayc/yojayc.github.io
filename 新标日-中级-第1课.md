## 课文

変化（へんか）が早（はや）くて、私たちもついていけない<span style="color:red">くらい</span>です
变话太快，甚至连我们都赶不上了呢

涙（なみだ）が出（で）る<span style="color:red">ほど</span>、うれしかったです
高兴得眼泪都流出来了

JR（ジェーアール）<span style="color:red">とは</span>、Japan（ジャパン） Railways（レールwウェイズ）の略（りゃく）です
JR 是 Japan Railways 的简称

世界（せかい）で初（はじ）めて実用（じつよう）<span style="color:red">化</span>（か）されたリニアモータカーも国営（こくえい）です
世界上首次实现商业运营的磁悬浮列车也是国营的

## 语法

### 〜くらいです [强调程度]

〜くらい / 〜ぐらい 表示[**大概的数量**](https://yojayc.github.io/2026/01/25/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC13%E8%AF%BE/#%E3%81%8F%E3%82%89%E3%81%84-%E3%81%90%E3%82%89%E3%81%84)

**强调某种状态达到何种程度（高、低）的表达方式（口语）：**

- 小句（简体形）+ くらい / ぐらい + 1形容词 / 2形容词 / 动词 / です

- 〜くらい / 〜ぐらい + <span style="color:red">の</span> + 名词

変化が早くて、私たちも　ついていけない　くらいです [高程度]

これは中学校（ちゅうがっこう）のテストですが、大人（大人）でも　分（わ）からない　くらい　難（むずか）しい問題（もんだい）です [高程度]
这虽是中学的考试题目，却是连大人都答不上来的难题

この辺（へん）は、静（しず）か過（す）ぎて　寂（さび）しい　くらいです [高程度]
这一带过于安静，甚至有几分寂寥

財布（さいふ）を忘（わす）れましたが、ジュースを買（か）うくらい　**の**お金（金）は持（も）っています [低程度]
我忘了带钱包，但买瓶饮料的钱还有

**高程度：**

忙（いそが）しくて、新聞（しんぶん）を読（よ）む暇（ひま）もないくらいです
忙到连看报纸的时间也没有

口（くち）から火（ひ）が出（で）るくらいの辛（から）さです
辣到嘴里喷火的程度

**低程度：**

食事（しょくじ）の前（まえ）に、手（て）ぐらい洗（あら）いなさい
吃饭前请至少洗手

名前（なまえ）を書（か）くくらいはできるだろう
至少写个名字你是会的吧

### 〜ほど

**小句（简体形）+ ほど** 也是强调程度的表达方式，主要用于**书面语**，只能强调**高程度**

涙（なみだ）が出（で）るほど、嬉（うれ）しかったです
高兴得眼泪都流出来了

昨日（きのう）はとても疲（つか）れました。最後（さいご）は　立（た）っていられないほど　でした [〜てられない 无法一直保持某种状态]
昨天太累了，到最后甚至站都站不住了

首都（しゅと）開発計画（かいはつけいかく）は、面白（おもしろ）いほど　うまく進（すす）んだ
首都发展规划，进展出奇地顺利

**〜ほど强调的程度更形象更夸张**

この携帯（けいたい）は高（たか）くて、目玉（めたま）が出るほどだ
这个手机特别贵，贵到眼球都要跳出来了

それは喉（のど）から手（て）が出（で）るほど欲（ほ）しいです
特别想要那个东西

今朝な地震はビルが倒（たお）れるほどの揺（ゆ）れだった
今天早上的地震大到大楼都要倒塌了

**〜ほど 惯用句**

やりたいことは　山（やま）ほどある
想做的事情特别特别多

時間（じかん）が　腐（くさ）るほど　あります
时间特别多

**没有什么比～更～：**

- 〜くらい　〜はない

- 〜ほど　〜はない

母（はは）ほど偉（えら）い人（ひと）はいない
没有什么人比母亲更伟大

真実（しんじつ）ほど残酷（ざんこく）なものはない
现实更加残酷

**くらい 有超过该程度的意思、ほど不超过**

お前ほどバカじゃないよ
我没你那么笨

今回（こんかい）のテストはそれほど難（むずか）しくなかった
这次考试没那么难

### 〜とは

**〜とは** 是 **〜というのは** 的省略形式。用于提出某个名称，并就其进行说明或询问

JRとは、Japan Railwaysの略です

宮保蝦仁（ゴンバオシャーレン）というのは、エビとピーナツと唐辛子（とうがらし）を炒（いた）めた料理（りょうり）です
宫保虾仁是用虾仁、花生米和辣椒炒出来的一道菜

**〜とは〜のことで** 用于下定义

おせち料理とは、正月（しょうがつ）に食（た）べるお祝（いわ）いの料理のことです
所谓“おせち料理”，就是过年时吃的庆贺新年的饭菜

**〜とは 作为结尾的时候，表示感叹，没想到**，口语用 **〜なんて**

こんなところで君（くん）に　会（あ）えるとは
没有想到在这种地方遇见你

警察（けいさつ）なのに、強盗（ごうとう）を見（み）て逃げる　なんて
身为一条狗渣，看到强盗竟然逃跑了

**〜というものは** 前接事物**名词**，用于说明该事物的本质特征（加入自己感慨感悟）

**〜ということは** 前接行为**动词**，用于说明该行为的本质或性质特征

<span style="color:red">人間</span>（にんげん）というのものは一人（ひとり）で生（い）きていくことはできません
人是不能一个人独立生存下去的生物

<span style="color:red">幸運</span>（こううん）というものはいつ訪（おとず）れるか分（わ）かりません
幸运何时到来是不可预知的

<span style="color:red">子供（こども）を育（そだ）てる</span>ということは、とても楽しいことです

一人で<span style="color:red">生活（せいかつ）する</span>ということは、大変（たいへん）なことです
一个人独自生活是很难的事情

**〜というものは 口语表达为 ってものは、っていうものは**

愛（あい）っていうものはガラス細工（さいく）みたいに壊（こわ）れやすいものです
爱这东西就像玻璃制品易碎

幸（しあわ）せというものは人から与（あた）えられるものではなく、自分の力（ちから）で掴（つか）むものだ
幸福并不是别人给予你的，而是用自己的双手抓住的

**小句 + ということは** 根据发生的事件推测该事件发生的原因或归纳出符合逻辑的结果 [逻辑性解释说明] 

彼（かれ）が怒（おこ）ったということは、きっとひどいことを言（い）われたのでしょう
他发火了，大概是因为有人对他说了非常过分的话吧

車（くるま）に五人（ごにん）か乗（の）れないということは、誰（だれ）かがバスで行（い）かなければならないということです
车上只能乘坐5人，也就是说必须有人乘公交车去

### ～化

**名词 + 化** 表示改变成该名词表示的状态，多前接**汉语**词和**外来**词。可以作**名词**，也可以后续 する 作**三类动词**

世界（せかい）で初（はじ）めて実用化（じつようか）されたリニアモーターカーも国営（こくえい）です
世界上首次实现商业运营的磁悬浮列车也是国营的

この村（むら）では近代化（きんだいか）が進（すす）み、高層（こうそう）ビルが建設（けんせつ）されています
这个村的现代化水平不断提高，正在修建高层建筑 

### 〜的

**名词 + 的** 表示带有～性质的、有关的～。相当于<sapn style="color:red">二类形容词</sapn>

これは中国（ちゅうごく）の伝統的（でんとうてき）な文化（ぶんか）です
这是中国传统的文化

科学的（かがくてき）な知識（ちしき）を学（まな）びましょう
我们来学一下有关科学的知识
