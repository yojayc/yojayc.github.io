## 课文

お荷物（にもつ）は私（わたくし）が<span style="color:red">お持（も）ちします</span>
您的行李我来拿

黄（こう）教授（きょうじゅ）に論文（ろんぶん）を<span style="color:red">見（み）ていただきました</span>
我请黄教授给我看了论文

明日、私がそちらへ　伺います　[行く]

中国（ちゅうごく）の北京（ペキン）から　参（まい）りました　[来る]
我来自中国北京

コピーは私が　いたします　[する]

黄教授に論文を見　ていただきました　[〜てもらう]

案内状（あんないじょう）、拝見（はいけん）いたしました　[見る]
开业通知，我们已经拜读了

### <span style="color:red">お</span> + 一类动词/二类动词（<span style="color:red">连用形</span>）します / <span style="color:red">ご</span> + <span style="color:red">三类动词</span>汉字部分 します

是自谦语中**用得最多**的一种表达形式

一类动词和二类动词连用形**只有一个音节的动词**，三类动词 **来（き）ます、します** 不能用这种形式

お荷物は私が　お持ちします　[持ちます]

昨日（さくじつ）メールを　お送（おく）りした　んですが　[送ります]
昨天，我给您发了邮件
ーー　ええ、拝見（はいけん）しました。先（さき）ほど返事（へんじ）を　お出（だ）ししました　[出します]
—— 是的，已经拜读了。刚刚给您发了回信

明日（みょうにち）の午後（ごご）、ご連絡（れんらく）します　[連絡します]
明天下午我跟您联系

**更礼貌**的用法是将 **お + 一类动词/二类动词（连用形）します / ご + 三类动词汉字部分 します** 中的 **<span style="color:red">します</span>** 改为 **<span style="color:red">いたします</span>**

お荷物は私が　お持ちいたします

明日の午後、ご連絡いたします

### 动词 ていただけますか　[请求]

**～ていただ<span style="color:red">き</span>ます** 用于请求对方为自己做某事时变成 **～ていただ<span style="color:red">け</span>ますか**。后者是前者的[**可能**](https://yojayc.github.io/2026/03/24/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC38%E8%AF%BE/#%E5%8F%AF%E8%83%BD%E5%BD%A2%E5%BC%8F)形式

**～ていただけますか** 更礼貌的形式是 **～ていただけませんか**

いただく　ー　いただきます（ます形）　ー　いただける（可能形）　ー　いただけます（ます形）　ー　いただけますか / いただけませんか

この服（ふく）、ちょっと小（ちい）さいので、取（と）り替（か）えていただけますか / いただけませんか
这件衣服小了点儿，能给我换一下吗？

教（おし）えていただけますか / いただけませんか
您能教我一下吗？

### 动词（さ）せていただきます

表示**请允许我做某事**，在**动词[使役形式](https://yojayc.github.io/2026/04/02/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC43%E8%AF%BE/#%E4%BD%BF%E5%BD%B9%E5%BD%A2%E5%BC%8F)** て形 后加 いただきます

使用**频率较高**，自谦程度高于 [**お / ご～します**](#お-一类动词-二类动词（连用形）します-ご-三类动词汉字部分-します)

早速（さっそく）、資料（しりょう）を届（とど）けさせていただきます
我这就把资料给您送过去

明日（みょうにち）、もう一度（いちど）連絡（れんらく）させていただきます
请允许我明天再跟您联络一次

> 更礼貌的表达形式为 **～（さ）せていただけますか** 和 **～（さ）せていただけませんか**

頭（あたま）が痛（いた）いんですが、帰（かえ）らせていただけますか
我头疼，能允许我回去吗？

来週（らしゅう）の水曜日（すいようび）、休（やす）ませていただけませんか
下个星期三，我想休息一下，行吗？

### ございます / 名 でございます

**あります** 更礼貌的说法是 **ございます**

**～です** 更礼貌的说法是 **～でございます**

すみません、この近（ちか）くにコンビニがありますか
请问，这附近有便利店吗？
ーー　はい、ございます。ホテルの前（まえ）の道（みち）を右（みぎ）に行（い）かれますと、左（ひだり）にございます
——— 有。沿着宾馆前面的路向右走，便利店在左边

社長（しゃちょう）、1（ひと）つお伺（うかが）いしたいことが　ございます
经理，我有一个问题向您请教一下

はい、JC（ジェーシー）企画（きかく）上海（シャンハイ）事務所（じむしょ）　でございます
喂，您好。这里是JC策划公司上海事务所

あのう、お客様（きゃくさま）、おつり　でございます
对不起，这是找您的钱

## 表达及词语讲解

### 动词外的敬语形式

#### 尊他语

在名词和形容词前加 **お** 或 **ご**。日语固有词前一般加 **お**，汉字词前一般加 **ご**

**名词**：

お食事（しょくじ）[饭]、お荷物（にもつ）[行李]、お手紙（てがみ）[信]、お客様（きゃくさま）[客人]

ご利用（りよう）[利用]、ご家族（かぞく）[家人]

> お客様 中的 **様** 也表示尊敬

**形容词**：

お忙（いそが）しい [忙]、お元気（げんき）[精神，精力充沛]、お若（わか）い [年轻]、お暇（ひま）[闲暇]

ご迷惑（めいわく）[麻烦，烦扰]、ご親切（しんせつ）[亲切;热心;好意]、ご熱心（ねっしん）[热心;热情]

### 明日 和 昨日

明日 有 **あした** [最随便]、**あす** [较郑重]、**みょうにち** [郑重程度最高，商业用语] 三种读法

昨日 有 **きのう** 和 **さくじつ** [较郑重] 两种读法

### 何（なん）のお構（かま）いもしませんでした [寒暄语]

客人离开时，主人说的招待不周，表示没能好好招待，非常抱歉的心情

也可以说成 **何のお構いもできませんでした**

### 右（みぎ）も左（ひだり）も分（わ）かりません

表示新到一个地方，对周围的情况一点儿也不了解

### おります / 〜ております

**います** 更谦和的说法是 **～おります**

**ています** 更谦和的说法是 **～ております**
