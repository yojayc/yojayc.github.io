课文

変化（へんか）が早（はや）くて、私たちもついていけないくらいです
变话太快，甚至连我们都赶不上了呢

涙（なみだ）が出（で）るほど、うれしかったです
高兴得眼泪都流出来了

JR（ジェーアール）とは、Japan（ジャパン） Railways（レールウェイズ）の略（りゃく）です
JR 是 Japan Railways 的简称

世界（せかい）で初（はじ）めて実用（じつよう）化（か）されたリニアモータカーも国営（こくえい）です
世界上首次实现商业运营的磁悬浮列车也是国营的

语法

〜くらいです [强调程度]

〜くらい / 〜ぐらい 表示大概的数量

强调某种状态达到何种程度（高、低）的表达方式（口语）：

  • 小句（简体形）+ くらい / ぐらい + 1形容词 / 2形容词 / 动词 / です

  • 〜くらい / 〜ぐらい + の + 名词

変化が早くて、私たちも　ついていけない　くらいです [高程度]

これは中学校（ちゅうがっこう）のテストですが、大人（大人）でも　分（わ）からない　くらい　難（むずか）しい問題（もんだい）です [高程度]
这虽是中学的考试题目，却是连大人都答不上来的难题

この辺（へん）は、静（しず）か過（す）ぎて　寂（さび）しい　くらいです [高程度]
这一带过于安静，甚至有几分寂寥

財布（さいふ）を忘（わす）れましたが、ジュースを買（か）うくらい　のお金（金）は持（も）っています [低程度]
我忘了带钱包，但买瓶饮料的钱还有

高程度：

忙（いそが）しくて、新聞（しんぶん）を読（よ）む暇（ひま）もないくらいです
忙到连看报纸的时间也没有

口（くち）から火（ひ）が出（で）るくらいの辛（から）さです
辣到嘴里喷火的程度

低程度：

食事（しょくじ）の前（まえ）に、手（て）ぐらい洗（あら）いなさい
吃饭前请至少洗手

名前（なまえ）を書（か）くくらいはできるだろう
至少写个名字你是会的吧

〜ほど

小句（简体形）+ ほど 也是强调程度的表达方式，主要用于书面语，只能强调高程度

涙（なみだ）が出（で）るほど、嬉（うれ）しかったです
高兴得眼泪都流出来了

昨日（きのう）はとても疲（つか）れました。最後（さいご）は　立（た）っていられないほど　でした [〜てられない 无法一直保持某种状态]
昨天太累了，到最后甚至站都站不住了

首都（しゅと）開発計画（かいはつけいかく）は、面白（おもしろ）いほど　うまく進（すす）んだ
首都发展规划，进展出奇地顺利

〜ほど强调的程度更形象更夸张

この携帯（けいたい）は高（たか）くて、目玉（めたま）が出るほどだ
这个手机特别贵，贵到眼球都要跳出来了

それは喉（のど）から手（て）が出（で）るほど欲（ほ）しいです
特别想要那个东西

今朝な地震はビルが倒（たお）れるほどの揺（ゆ）れだった
今天早上的地震大到大楼都要倒塌了

〜ほど 惯用句

やりたいことは　山（やま）ほどある
想做的事情特别特别多

時間（じかん）が　腐（くさ）るほど　あります
时间特别多

没有什么比～更～：

  • 〜くらい　〜はない

  • 〜ほど　〜はない

母（はは）ほど偉（えら）い人（ひと）はいない
没有什么人比母亲更伟大

真実（しんじつ）ほど残酷（ざんこく）なものはない
现实更加残酷

くらい 有超过该程度的意思、ほど不超过

お前ほどバカじゃないよ
我没你那么笨

今回（こんかい）のテストはそれほど難（むずか）しくなかった
这次考试没那么难

〜とは

〜とは 是 〜というのは 的省略形式。用于提出某个名称，并就其进行说明或询问

JRとは、Japan Railwaysの略です

宮保蝦仁（ゴンバオシャーレン）というのは、エビとピーナツと唐辛子（とうがらし）を炒（いた）めた料理（りょうり）です
宫保虾仁是用虾仁、花生米和辣椒炒出来的一道菜

〜とは〜のことで 用于下定义

おせち料理とは、正月（しょうがつ）に食（た）べるお祝（いわ）いの料理のことです
所谓“おせち料理”，就是过年时吃的庆贺新年的饭菜

〜とは 作为结尾的时候，表示感叹，没想到，口语用 〜なんて

こんなところで君（くん）に　会（あ）えるとは
没有想到在这种地方遇见你

警察（けいさつ）なのに、強盗（ごうとう）を見（み）て逃げる　なんて
身为一条狗渣，看到强盗竟然逃跑了

〜というものは 前接事物名词，用于说明该事物的本质特征（加入自己感慨感悟）

〜ということは 前接行为动词，用于说明该行为的本质或性质特征

人間（にんげん）というのものは一人（ひとり）で生（い）きていくことはできません
人是不能一个人独立生存下去的生物

幸運（こううん）というものはいつ訪（おとず）れるか分（わ）かりません
幸运何时到来是不可预知的

子供（こども）を育（そだ）てるということは、とても楽しいことです

一人で生活（せいかつ）するということは、大変（たいへん）なことです
一个人独自生活是很难的事情

〜というものは 口语表达为 ってものは、っていうものは

愛（あい）っていうものはガラス細工（さいく）みたいに壊（こわ）れやすいものです
爱这东西就像玻璃制品易碎

幸（しあわ）せというものは人から与（あた）えられるものではなく、自分の力（ちから）で掴（つか）むものだ
幸福并不是别人给予你的，而是用自己的双手抓住的

小句 + ということは 根据发生的事件推测该事件发生的原因或归纳出符合逻辑的结果 [逻辑性解释说明]

彼（かれ）が怒（おこ）ったということは、きっとひどいことを言（い）われたのでしょう
他发火了，大概是因为有人对他说了非常过分的话吧

車（くるま）に五人（ごにん）か乗（の）れないということは、誰（だれ）かがバスで行（い）かなければならないということです
车上只能乘坐5人，也就是说必须有人乘公交车去

～化

名词 + 化 表示改变成该名词表示的状态，多前接汉语词和外来词。可以作名词，也可以后续 する 作三类动词

世界（せかい）で初（はじ）めて実用化（じつようか）されたリニアモーターカーも国営（こくえい）です
世界上首次实现商业运营的磁悬浮列车也是国营的

この村（むら）では近代化（きんだいか）が進（すす）み、高層（こうそう）ビルが建設（けんせつ）されています
这个村的现代化水平不断提高，正在修建高层建筑

〜的

名词 + 的 表示带有～性质的、有关的～。相当于二类形容词

これは中国（ちゅうごく）の伝統的（でんとうてき）な文化（ぶんか）です
这是中国传统的文化

科学的（かがくてき）な知識（ちしき）を学（まな）びましょう
我们来学一下有关科学的知识

语法与表达

搭话、打招呼

提问 回答
すみません、〜 はい？
あのう、すみません。〜 何（なん）でしょうか
ちょっと、すみません。〜 あっ、どうも
あのう、失礼（しつれい）ですが、〜  

对初次见面的尊长使用 “ちょっと” 很失礼

自动词与他动词

日语中存在相应的自他动词时，不能用无生命体充当主语

先ほど

先（さき）ほど [书面语] 是 さっき [口语] 的礼貌说法，意思相同

先ほどの方

日语中省略掉动词，采用 名词 + の + 名词 的形式也很常见

助かりました

单词 假名 声调 释义
助ける たすける 3 帮助，援助
助かる たすかる 3 获得帮助；得救

感じがします

〜感じがします 表示有那样的感觉、有那样的印象

感じます 多表示直接的感受

〜へは 与 〜は

助词 例句
が + は = は この駅の駅弁は美味しいです
を + は = は お酒は飲みません
へ + は = は / へは 京都は（へは）行きませんでした
と + は = とは 李さんとはどこで待ち合わせましたか
から + は = からは 外からは中に誰がいるのか分かりません

転勤になった

〜になります 表示不是出于本人的意愿

すっかり 与 ずいぶん

都可以表示变化的程度

すっかり 表示 “完全、彻底地”，不能用于表示变化的量

ずいぶん 表示 “相当、非常”，可以表示变化的量，可以修饰形容词

ついていけない

ついていけない 是动词 ついていきます 的可能形式 ついていけます 对应的否定形式 ついていけません 的简体形式

本意是 “跟不上脚步”，引申为难以有相同的想法或行动

〜てきました

表示变化的开始，或不断朝着某个结果进展的状态

AはBの一（ひと）つです

表示A属于B范围当中的一种

は / も 是提示助词，可以互换

中国はアジアの国の一つです。日本もアジアの国の一つです

〜結（むす）ぶ

結ぶ 本意是 “结绳”，后引申为 “连结、建立关系” 等

用于句尾时，使用 “結んでいます” 表示状态

大～

读作 だい 时，后续一般为两个字的汉语词，如 “大事件(だいじけん)、大地震（だいじしん）、大成功（だいせいこう）”

读作 たい 时，“大” 一般已经成为词语的一部分，不能分离，如“大変（たいへん）、大切（たいせつ）”

读作 おお 时，后续一般为日语固有词，如“大金持ち（おおかねもち）、大雨（おおあめ）、大汗（おおあせ）”

不是所有名词前都可以加 “大”，如 “大感谢、大进步” 在日语中是不说的

大規模（だいきぼ）是二类形容词，用 大規模な 的形式修饰名词

だい〜：

大事件、大規模（だいきぼ）、大感激（だいかんげき）

おお〜：

大風（おおかぜ）、大昔（おおむかし）、大笑い（おおわらい）

また

表示并列，“另外”，常用于句首

上（のぼ）り、下（くだ）り

上り、下り 反映了首都东京处于高点，其他地区较低的想法

上りの電車 指开往东京方向的列车

下りの電車 指从东京开往周边地区的列车

そこで、それで

それで 表示所以（前述内容是后述内容的原因）

そこで 表示于是（针对前文的状况，该怎么做，出现了什么）
