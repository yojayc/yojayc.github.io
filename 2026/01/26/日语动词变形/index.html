日语动词变形

三类动词（カ变、サ变）—— 不规则动词

する　ー　して / した

来（く）る　ー　来（き）て / 来（き）た

二类动词（上一段、下一段）

る　变成　て / た

食（た）べる　ー　食べて / 食べた

調（しら）べる　ー　調べて / 調べた

一类动词（五段动词）

规则 示例
す　ー　して / した 話（はな）す　ー　話して / 話した
く　ー　いて / いた 磨（みが）く　ー　磨いて / 磨いた
ぐ　ー　いで / いだ 急（いそ）ぐ　ー　急いで / 急いだ
う、つ、る　ー　って / った 洗（あら）う　ー　洗って / 洗った
  待（ま）つ　ー　待って / 待った
  やる　ー　やって / やった
ぶ、む、ぬ　ー　んで / んだ 遊（あそ）ぶ　ー　遊んで / 遊んだ
  読（よ）む　ー　読んで / 読んだ
  死（し）ぬ　ー　死んで / 死んだ
行（い）く　ー　行って / 行った  
　　　

示例

勉強（べんきょう）する、見（み）る、帰（かえ）る、行（い）く

3、2、1、1

勉強して、見て、帰って、行って


好（この）む、考（かんが）える、呼（よ）ぶ、急（いそ）ぐ

1、2、1、1

好んで、考えて、呼んで、急いで
