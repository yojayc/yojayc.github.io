新标日-初级-第46课

第46课这个软软乎乎的，好像真皮一样

动词

ます形 假名 原形 假名 释义
済みます すみます 済む すむ 结束，完了
積もります つもります 積もる つもる 堆积;积攒
過ごします すごします 過ごす すごす 度过;生活
ほほえみます   ほほえる   微笑
延ばします のばします 延ばす のばす 延长
済ませます すませます 済ませる すませる 搞完，办完
枯れます かれます 枯れる かれる 枯萎;干燥
発送します はっそうします 発送する はっそうする 发送，寄送
緊張します きんちょうします 緊張する きんちょうする 紧张

课文

これは柔（やわ）らかくて、まるで本物（ほんもの）の毛皮（けがわ）のようです
这个软软乎乎的，好像真皮一样

この着物（きもの）はいかにも日本（にほん）らしい柄（がら）ですね
这件和服的花纹是典型的日本式

明日（あした）の九時（くじ）までにこの書類（しょるい）を完成（かんせい）させなければなりません
明天9点之前必须完成这份文件

私が留学（りゅうがく）している間（あいだ）に、家（いえ）の周（まわ）りもずいぶん変（か）わりました
我留学期间，我家那一带发生了很大的变化

语法

よう / みたいです [比喻]

除了表示推测，还可以用来表示比喻，多与副词 “まるで” 呼应使用（翻译成好像……宛如……）

名词 のようです

动词（简体形）ようです

名词 みたいです

动词（简体形）みたいです

これは柔らかくて、まるで本物の　毛皮のようです

まるでヨーロッパの町並（まちな）みを　見（み）ているようですね
真像看到了欧洲的街道

この野菜（やさい）、まるで　果物（くだもの）みたいですね
这种菜好像水果一样

这个景色真像看到了宇宙
この景色（けしき）、まるで宇宙（うちゅう）を見ているようです

除了用于句尾外，常以 ～ような/みたいな + 名词 或 ～ように/みたいに + 动词/形容词 的形式修饰名词、动词或形容词

まるで　テレビのドラマのような出来事（できごと）　ですね
真像是电视剧里发生的事情一样

鳥（とり）のように　空（そら）を飛（と）んで　みたいなあ
我想像小鸟一样在天空飞翔

先生（せんせい）みたいに上手（じょうず）に日本語（にほんご）が話（はな）せるようになりたいです
我想像老师那样流利地说日语

Aらしい + B [典型性]

除了表示推测，らしい 接在 A 后表示 B 具有 A 的典型特征，多与 いかにも 呼应使用

この着物はいかにも　日本らしい柄ですね

今日（きょう）は　春（はる）らしい天気（てんき）です
今天像是春天的天气

南京路（なんきんろ）はいかにも　上海（シャンハイ）らしい所（ところ）ですよ
南京路是上海最具代表性的地方

A 和 B 是同一个名词时，表示 “真正意义上的～、像样的～”

仕事（しごと）らしい仕事　はしたことがありません
我没干过像样的工作

このごろ忙（いそが）しくて、食事（しょくじ）らしい食事　をしていません
最近很忙，没正经吃过一顿饭

时间までに

時間（じかん）+ まで 表示动作持续的最后时间

時間 + までに 表示某种事态发生或完成的截止时间

明日の九時までに　この書類を完成させなければなりません

今月（こんげつ）の末（すえ）までに、新（あたら）しい家（いえ）を見（み）つけたいです
在月底之前我想找到一处新房子

来月（らいげつ）の初（はじ）めまでには　事務所（じむしょ）開設（かいせつ）の準備（じゅんび）を済（す）ませたいです
下个月初之前，要完成开设事务所的所有准备工作

下个月前我想吃一次旋转寿司
今月の末までに、回転（かいてん）寿司を食べたいです

名词 + の / 动词（简体形）[〜ている間/間に] + 間（あいだ）/ 間に

～間 表示某种行为或事态持续的时间段。多与副词 ずっと 一起使用

夏休（なつやす）みの間、ずっとアルバイトをしていました
暑假期间一直在打工

夏休みの間に、アルバイトをしました
暑假期间打过工


学校（がっこう）が休みの間、アルバイトをするつもりなんだ
我打算在学校放假的时候打工

部屋（へや）の中（なか）を見（み）ている間、ずっと外（そと）で車（くるま）の音（おと）がしていましたよ
我们在看房间里面的时候，外面一直是过车的噪音

私が留学している　間に、家の周りもずいぶん変わりました

子供（こども）が寝（ね）ている間に、部屋を掃除（そうじ）しました
趁孩子睡觉的时候，我打扫了房间

味（あじ）がします [味觉] / においがします [嗅觉] / 声（こえ）/ 音（おと）がします [听觉] / 感（かん）じがします [感觉]

この牛乳（ぎゅうにゅう）、変（へん）な味がするけど、いつ買（か）ったの？
这牛奶有股怪味，你什么时候买的？

这个果汁有西瓜的味道（嗅觉）
このジュース、スイカ　のにおいがします

外面有阿飞老师的声音
教室（きょうしつ）の外（そと）で、トビ先生の声がします

名词のような味がします [味觉]

名词のようなにおいがします [嗅觉]

この野菜（やさい）、レモンのような　味がしますね
这菜有股柠檬的味道

このスープは、日本（にほん）のみそ汁（しる）のような　においがします
这汤有股日本酱汤的味道

外で子供（こども）の声がします
外面有孩子们的声音

ここには昔（むかし）来（き）たことがある感じがします
我觉得以前来过这里

表示身体状况的词语也可以加在 がします 前使用，如 頭痛（ずつう）がします [头疼]、寒気（さむけ）がします [感到发冷]

さっきから、ちょっと吐（は）き気（け）がするんです
从刚才开始，觉得有点儿恶心

表达及词语讲解

済（す）ませたい

他动词，表示 “结束某件事”

对应的自动词为 済みます，表示事情结束

〜んじゃないですか [婉转陈述意见]

与 ～のではないのでしょうか 意思基本相同，但礼貌程度略低

いかにも

表示特别像某种样子或浓厚地具有某种性质，多与 ～らしい 呼应使用
