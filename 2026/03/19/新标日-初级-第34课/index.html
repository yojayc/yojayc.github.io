课文

壁（かべ）にカレンダーが掛（か）けてあります
墙上挂着挂历

お客（きゃく）さんが来（く）る前（まえ）に、部屋（へや）を掃除（そうじ）しておきます
客人来之间把房间打扫好

太田（おおた）さんは中国語（ちゅうごくご）で手紙（てがみ）を書（か）いてみました
太田先生试着用汉语写了一封信

日本（にほん）へ留学（りゅうがく）するために、お金（かね）をためています
我正在为去日本留学攒钱

语法

补助动词

动词て形 动词て形 动词 表示动作先后发生

补助动词 + てあります / ておきます / てみます 用于对前面的动词进行补充

他动词 てあります

表示看上去是现象，实际暗示是有人做的

壁にカレンダーが　掛け　てあります

窓（まど）が　開（あ）け　てあります
窗户开着

パスポートは　バッグの中（なか）に　入（い）れ　てあります
护照在包里放着

冰箱里放着可乐
コーラは　冷蔵庫（れいぞうこ）のなかに　入れ　てあります
冷蔵庫にコーラが入れてあります

壁にカレンダーが掛かっています [自動詞（じどうし）＋ている]
描述墙上挂着挂历的现象

壁にカレンダーが掛けてあります [他動詞（たどうし）＋てある]
暗示是我挂的挂历

动词 ておきます

表示事先做某事，预先做某事
　　　　　　　　　　　　　　　
お客さんが来る前に、部屋を　掃除し　ておきます　[おきます 表示还没打扫，おきました 表示打扫了]

明日（あした）は忙（いそが）しいです。今夜（こんや）はゆっくり　休（やす）ん　でおいてください
明天会很忙，今天晚上你要好好休息

訪問（ほうもん）する前（まえ）に、一度（いちど）　電話（でんわ）し　ておきます
拜访之前我准备先打个电话

帰（かえ）りの切符（きっぷ）を　買（か）っ　ておきますか
要事先买好回程票吗？
ーー　いいえ、買っ　ておかなくてもいいです
—— 不，不需要

我已经把行李整理好了
荷物（にもつ）を　整理（せいり）しておきました

动词 てみます

表示尝试做某事

太田さんは中国語で手紙を　書い　てみました

おいしそうですね。食べ　てみます
看上去很好吃啊，我来尝尝

それはおかしい。もう一度（いちど）　調（しら）べ　てみてください
真奇怪。你再调查一下

小李试着用手机发文件
李さんは　スマホで　ファイル / 書類（しょるい）を　送（おく）って　みます

小句1（基本形）ために，小句2 / 名词 + の + ために，小句 [目的]

ために 表示目的，前后两个小句的主语相同（人为、主动、有意识去做）

日本に　留学する　ために、お金をためています
に 强调最终到达。へ 只强调往哪边走，不强调到达

運動（うんどう）を　する　ために、新（あたら）しい靴（くつ）を買（か）いました
为了运动买了新鞋

林（はやし）さんは駅（えき）へ　行（い）く　ために、バスに乗（の）りました
林先生为了去电车站，乘了公共汽车

前接名词时，用 名词 + の + ために 的形式

留学（りゅうがく）　の　ために、日本語（にほんご）を勉強（べんきょう）しています
为了留学正在学习日语

健康（けんこう）　の　ために、毎日（まいにち）歩（ある）いています
为了健康每天步行

为了身体每天吃蔬菜
健康 / 体（からだ）のために、毎日野菜を食べています

ため 偏书面，正式场景用

表达及词语讲解

とっておきのお店（みせ）

とっておき 表示珍藏，私藏

とっておきの + 名词 表示为了某种场合或用途而特别留着的东西或场所

しっかり

课文中表示充分，充足，还有扎实、踏实、认真等意思

うっかり 表示疏忽;不留神;无意中

あちこち

表示多种场所或多个方向。本来形式是 あちらこちら

日语中只存在 那 在前，这 在后的说法

もったいない

表示没有做到物尽其用，人尽其能而感到非常遗憾
