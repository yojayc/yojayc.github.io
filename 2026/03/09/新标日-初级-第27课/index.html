新标日-初级-第27课

第27课我小时候，发生过大地震

动词

ます形 假名 原形 假名 释义
通います かよいます 通う かよう 上学，来往
集まります あつまります 集まる あつまる 聚，集合
踊ります おどります 踊る おどる 跳舞
要ります いります 要る いる 要
困ります こまります 困る こまる 为难，难办
たたきます   たたく   拍，敲，打
入れます いれます 入れる いれる 放入，放进
看病します かんびょうします 看病する かんびょうする 护理
けんかします   けんかする   吵架，打架
利用します りようします 利用する りようする 利用
相談します そうだんします 相談する そうだんする 商谈

课文

子供（こども）の時（とき）、大（おお）きな地震（じしん）がありました
我小时候，发生过大地震

映画（えいが）を見（み）る時（とき）、いつもいちばん後（うし）ろの席（せき）に座（すわ）ります
看电影时，我经常坐在最后的座位

李さんはテレビを見（み）ながら食事（しょくじ）をしています
小李正边看电视边吃饭

李さん、明日（あした）パーティーに行（い）くでしょう
小李，你明天参加联欢会吧？

语法

小句（简体形）+ 時

  • 名词 + の + 時 [子供の時]

  • 动词 + 時 [学校（がっこう）へ行（い）く時]

  • 一类形容词 + 時 [忙（いそが）しい時]

  • 二类形容词 + な + 時 [暇（ひま）な時]

子供の時、大きな地震がありました

日本（にほん）に行（い）く時、たくさんお土産（みやげ）を買（か）いました（要去日本，还没到）
去日本的时候买了许多礼物

日本に行った時、たくさんお土産を買いました
去了日本的时候买了许多礼物

忙しい時、家族（かぞく）みんなで仕事（しごと）をします
忙时全家人一起干活

暇な時、私は町（まち）で買い物（もの）をします
有空时我上街买东西

阿飞老师上课的时候，我偷偷地吃了一口面包
トビ先生が授業（じゅぎょう）をしている時、私はこっそりパンを食べました

表示一次性，个别的情况时，在 “時” 后加 “に”

この前（まえ）、友達（ともだち）が病気（びょうき）の時に、私が看病（かんびょう）しました
前些天朋友生病时我照看他了

友達が病気の時に、いつも私が看病しました　X

动 ながら　[ます形去掉ます]

表示同时进行两个动作，后面的动作是主要动作

　　　　テレビを見ます　食事をします
李さんはテレビを見ながら食事をしています

　　　　　考えます　　　　　　歩きます
そのことを考（かんが）えながら歩（ある）いていました
一边考虑着那件事一边走着

阿飞老师一边听音乐一边跑步
トビ先生音楽（おんがく）を聞きながらジョギングをしています

小句（简体形）でしょう　[确认]

确认对方比自己更为熟悉的事情，请对方告诉自己某种信息，读升调

用于对方和自己意见不同，或叮问对方，对降调

李さん、明日パーティーに行くでしょう [升调]

李さんは優（やさ）しくて、親切でしょう [升调]
小李又和蔼又热情吧

このお菓子（かし）まずいね
这个点心真难吃
ーー　えっ！まずい？おいしいでしょう　[降调]
—— 哎！难吃？好吃吧

お母（かあ）さん、私の日記（にっき）、見（み）たでしょう　[降调]
妈妈，你是不是看了我的日记？

音音桑，你喝了我的珍珠奶茶吧！
ネネさん、私のタピオカ、飲んだでしょう

说话人认为自己与听话人意见一致时用 “ね”

今日は暑いですね

今日は暑ぃでしょう　X

A：这道题好难啊（A觉得B也不会）
この問題（もんだい）は難（むずか）しいね

B：没这回事儿吧…
そんなことないけど/難しですか

动　ています [反复] [习惯]

动词て形 + います 除了表示和的用法之外，还可以表示反复或习惯性的动作。表示习惯性动作时可以用动词 “ます形” 代替

葉子（ようこ）さんはアルバイトをしながら学校（がっこう）に通（かよ）っています　[反复]
叶子边打工边上学

北京（ペキン）行（い）きの飛行機（ひこうき）は一時間（いちじかん）に一便（いちびん）飛（と）んでいます　[反复]
飞往北京的飞机每小时飞一架

私は毎日（まいにち）散歩（さんぽ）しています　[习惯]
我每天散步

私は毎日散歩します　[习惯]
我每天散步

表示习惯性动作的句子里使用 “先月（せんげつ）から” 等表示时间起点的形式，一般不用 “ます”

表示现在正在从事的工作，可以用 [表示职业的名词] + を + しています 的形式

私は高校（こうこう）の教師（きょうし）をしています
我在做高中教师

我是出租车司机
私はタクシーの運転手（うんてんしゅ）をしています

名で　[原因，理由]

で 加在名词后表示原因、理由

仕事（しごと）で、楊（よう）さんと会（あ）っていたんですよ
因工作我和杨先生见了面

名词　と会（あ）います

使用表示互动行为的动词如 “結婚（けっこん）します [结婚]” “けんかします [吵架]” 时只能用 “と”

会（あ）います [见] 可以用 “に”，也可以用 “と”，用 “と” 时含有互动的语感

仕事で、楊さんと会っていたんですよ

田中（たなか）さんは佐藤（さとう）さんと結婚しました
田中先生和佐藤女士结婚了

我和小王打了一架
私は王さんとけんかしました

在 AはBと〜 中A是主语。在 AとB（と）が〜 中，A 和 B 都是主语，と 可省略

田中さんと佐藤さん（と）が結婚しました

田中さんと佐藤さんは結婚しました

表达及词语讲解

学校（がっこう）に通（かよ）っています

通います 表示定期来往于同一个场所，与表示目的地的 “に” 一起使用

行きます 只表示从一个地方移动到另一个地方，没有往返的意思

大勢（おおぜい）

表示人多，只能用于人。不能做谓语

大勢の人
很多人

人が大勢います
有许多人

お年寄（としよ）り

お年寄り 是既有敬意又有亲切感的 “老人” 的说法，但不能直接称呼对方。可以用 “おじいさん（老爷爷）”　“おばあさん（老奶奶）” 称呼对方

そう言（い）えば

表示说起来……样

へえ

表示对听到的信息感到惊讶或钦佩
