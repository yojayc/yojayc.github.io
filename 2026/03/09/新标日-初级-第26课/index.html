课文

自転車（じてんしゃ）に二人（ふたり）で乗（の）るのは危（あぶ）ないです
骑自行车带人很危险

手紙（てがみ）を出（だ）すのを忘（わす）れました
我忘了寄信了

明日（あした）の朝（あさ）は大雨（おおあめ）になるでしょう
明天早晨会下大雨吧

森（もり）さんは今日（きょう）会社（かいしゃ）を休（やす）むかもしれません
森先生今天也许不来上班

语法

小句（动词简体形）+ の + は　形　です

动词小句加 “の” 使其名词化（形式体言），のは 连读。做主语时，谓语是一类形容词或二类形容词时，助词用“は”

自転車に二人で乗る　の　は　危ないです

パソコンで表（ひょう）を作（つく）る　の　は　楽（たの）しいです
用电脑制表是很愉快的

一个人看电影很寂寞
人（ひとり）で映画（えいが）を見る　の　は　寂（さび）しいです

谓语是 好（す）きです、嫌（きら）いです、上手（じょうず）です、下手（へた）です 等主观性强的形容词时，助词用“が”

李（り）さんは　絵（え）を書く　の　が　好（す）きですね
小李你喜欢画画啊！
ーー　はい、色鉛筆（いろえんぴつ）で　スケッチする　の　が　大好（だいす）きです
—— 是的，我特别喜欢用彩色铅笔素描

わたしは　自転車（じてんしゃ）に乗（の）る　の　が　下手（へた）です
我不擅长骑自行车

佐藤喜欢吃中国菜
佐藤さんは中華料理（ちゅうかりょうり）を食べる　の　が　好きです

好（す）きです 等表示好恶的形容词的对象作为话题使用时，用助词“は”

絵（え）をかく　のは　好きですか
画画你喜欢吗？
ーー　いいえ、絵をかくのは嫌（きら）いです
—— 不，画画我不喜欢

小句（动词简体形）+ の + を　动

动词小句加“の”的名词化形式还可以做宾语

手紙を出す　の/こと　を　忘れました

森（もり）さんが　発言（はつげん）する　の　を　聞（き）きました
听了森先生的发言

林（はやし）さんは　掃除（そうじ）する　の　を　やめました
林先生不做扫除了

李さんは　小野（おの）さんが料理（りょうり）を　作（つく）る　の　を　手伝（てつだ）います
小李帮小野女士做菜

名词化形式做宾语“の”和“こと”可以互换

句尾动词是 知（し）らせます[告诉]、伝（つた）えます[转告]、話（はな）します[说] 等表示语言行为的动词时，只能用“こと”

句尾动词是 見（み）ます[告诉]、聞（き）きます[转告] 等表示感知的动词或 手伝（てつだ）います[帮忙]、やめます[停止]、防（ふせ）ぎます[防御] 等表示直接作用于动作对象的动词时，只能用“の”

李さんは　森さんが　休（やす）む　こと　を　伝えました
小李转告了森先生请假的事情

李さんは　森さんが　車（くるま）から降（お）りる　の　を　見ました
小李看见森先生从车上下来了

李さんは　森さんが　荷物（にもつ）を運（はこ）ぶ　の　を　手伝います
小李帮森先生运行李

小句（简体形）でしょう [推测]

表示对不能断定的事情进行推测，常与“たぶん”呼应使用。

  • 动词和一类形容词简体形后接 でしょう

  • 二类形容词和名词将 “だ” 换成 “でしょう”

  • でしょう 的简体形（口语表达）是 だろう

明日の朝は大雨になる　でしょう

森さんは知（し）らない　でしょう
森先生不知道吧

この本（ほん）の値段（ねだん）は分（わ）かりませんが、たぶん高（たか）い　でしょう
我不知道这本书的价格，大概很贵吧

桜（さくら）の季節（きせつ）ですから、京都（きょうと）はたぶんにぎやか　でしょう
现在正是观赏樱花的季节，京都大概很热闹吧

今年（ことし）はたぶん豊作（ほうさく） だろう
今年大概是个丰收年吧

これはたぶんだれかの忘（わす）れ物（もの）　だろう
这大概是谁忘记的东西吧

会議（かいぎ）はもう終（お）わった　だろう
会议已经结束了吧

小句（简体形）かもしれません

表示有50%可能发生某事

森さんは 今日会社を休む かもしれません

あの二人（ふたり）は仲（なか）がいいですね
那两个关系很好啊
ーー　あの二人は結婚（けっこん）する かもしれませんね
—— 也许他们会结婚吧

クレジットカードを落（お）とした かもしれません
也许我把信用卡弄丢了

来週（らいしゅう）は暇（ひま）かもしれません
下个星期也许有空

明日（あした）は雪（ゆき）かもしれません
明天也许下雪

可以在同一动作的肯定形和否定形后并列使用两个 “かもしれません”

明日は雨（あめ）が降（ふ）るかもしれませんが、降（ふ）らないかもしれません
明天也许下雨，也许不下雨

表达及词语讲解

もしかしたら

口语形式，用于可能性较低的场合，一般与 “かもしれません”　“ではありませんか” 呼应使用

それで

表示前面句子的事态为后面句子事态的原因、理由

だから 也表示原因、理由，可以接祈使或推测的句子，それで 不行

つい

不知不觉，表示后悔、不愉快

あいさつ回（まわ）り

到新单位或搬家后，和隔壁打招呼
