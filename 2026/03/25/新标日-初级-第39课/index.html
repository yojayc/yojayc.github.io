第39课戴着眼镜看书

动词

ます形 假名 原形 假名 释义
見渡します みわたします 見渡す みわたす 眺望，远眺
続きます つづきます 続く つづく 继续，持续
なくなります   なくなる   完，丢失
育ちます そだちます 育つ そだつ 生长，成长
増えます ふえます 増える ふえる 增加，增多
連れます つれます 連れる つれる 带，领
欠席します けっせきします 欠席する けっせきする 缺席
輸入します ゆにゅうします 輸入する ゆにゅうする 进口，输入

课文

眼鏡（めがね）をかけて本（ほん）を読（よ）みます
戴着眼镜看书

道路工事（どうろこうじ）のために、道（みち）が込（こ）んでいます
因为施工，道路非常拥堵

李（り）さんは急（いそ）いで帰（かえ）っていきました
小李急急忙忙地回去了

去年（きゅねん）、日本（にほん）で歌舞伎（かぶき）を見（み）てきました
去年，在日本看了歌舞伎

语法

小句1 て / ないで + 小句2

表示进行小句2的动作主体处于小句1的状态时，小句1句尾使用动词て形或动词ない形 + で

眼鏡をかけて　本を読みます

手（て）を挙（あ）げて　横断歩道（おうだんほどう）を渡（わた）ります
举着手过人行横道

今日（きょう）は傘（かさ）を持（も）た　ないで / ずに　出（で）かけました
今天没有带伞就出门了

戴着帽子跑步
帽子（ぼうし）をかぶって　グラウンドで走（はし）ります

〜て 的前后主语可以不一致：

  • 表示动作相继发生

  • 表示并列

  • 表示原因、理由

〜て 表示动作主体状况时，前后主语必须一致

区分 ないで なくて

有因果关系用なくて，其他时候用ないで

バスが　なかなか　[来（こ）ないで、来（こ）なくて]、イライラしています
因为公交车一直不来，所以心情特别烦

小林（おばやし）さんは　[勉強しないで、勉強しなくて]、ずっと女（おんな）の子（こ）と話（はな）している
小林不学习，一直在和女孩聊天

形容词、名词只有なくて

ユニちゃんは　[賢（かしこ）くないで、賢くなくて]、わがままです
youni不聪明，很任性

今回（こんかい）のテストは　[簡単（かんたん）じゃないで、簡単じゃなくて]、がっかりしています
这次的考试很难，我很失望

小句1（简体形）ために，小句2

表示小句1是小句2的原因、理由。小句1表示非意志性的事态或状态（一般是不情愿的事情，不是好事）。ために不能作为句尾出现

把ため看成名词：

  • 動詞（る、た）＋ために

  • 1形容詞＋ために

  • 2形容詞＋な＋ために

  • 名詞＋の＋ために

事故（じこ）が　あったために、電車（でんしゃ）が遅（おく）れたんです
由于发生事故，电车晚点了

明日（あした）マラソン大会（たいかい）が　あるために、この道路（どうろ）は通行禁止（つうこうきんし）になります
明天有马拉松大会，所以这条路禁止通行

この村（むら）は交通（こうつう）が　不便（ふべん）なために、住（す）んでいる人（ひと）が少（すく）ないです
这个村子因为交通不便，居住的人很少

道路工事（どうろこうじ）のために、道（みち）が込（こ）んでいます
因为施工，道路非常拥堵

因为小林很丑，所以被甩了
小林（こばやし）君（くん）はとても　醜（みにく）い / ブサイクなために

表示原因、理由时，与〜から、〜ので、〜て 相比，〜ために 主要用于表示不情愿的事情的原因

〜から、〜ので/んで 和 〜て 可以用于结尾，〜ために不行

动　ていきます / きます

〜ていきます / きます 表示以说话人为起点的移动的方向

接在 帰る、歩く、走る、泳ぐ、飛ぶ、乗る 等表示移动的动词后

〜ていきます 表示由近及远的移动，去

〜てきます 表示由远及近的移动，来

李さんは急いで帰っていきました

ずいぶんたくさん人（ひと）が乗（の）ってきましたね
上来这么多人啊！


〜ていきます / きます 接在一部分动词后表示一种独立的动作，分别表示两个相继发生的动作

去年、日本で歌舞伎を見てきました

疲（つか）れているんでしょう？ちょっと休んでいきなさい
累了吧？歇会儿再走吧


〜ていきます / きます 表示一种独立的动作

て 前的动词表示动作主体的状况

子供（こども）を病院（びょういん）に連（つ）れていきます
带孩子去医院

明日（あした）お弁当（べんとう）を持ってきてください
明天请带盒饭来


〜てきます 表示循环性的动作

口语中，常用于描述说话人或其他人去了一个地方，然后又回到原来的场所

ちょっと手紙（てがみ）を　出（だ）してきます
我去寄封信就回来

ちょっと手（て）を　洗（あら）ってきます。ここで待（ま）っていてください
我去洗洗手就来。请在这里等我

表达及词语讲解

も [超出预想]

助词 も 有时可用来表示数量比事先预想的或应有的多时的心情

遠（とお）く

一类形容词词尾 い 变 く 做副词，但 遠く 和 近（ちか）く 用作名词，分别表示 “远处” 和 “近处”

絶対（ぜったい）

表示一定、绝对。既可以修饰否定的内容，也可以修饰肯定的内容

絶対に 意思不变

せっかく

表示特意，好不容易

当别人劝说自己做某事时：

  • 接受其好意时用 せっかくですから

  • 拒绝时用 せっかくですが
