课文

優勝（ゆうしょう）すれば、オリンピックに出場（しゅつじょう）することができます
如果获胜，就能够参加奥运会

天安門（てんあんもん）へ行（い）くなら、地下鉄（ちかてつ）が便利（べんり）ですよ
如果去天安门，坐地铁很方便

映画（えいが）でも見（み）に行（い）きませんか
去看电影怎么样？

パーティーで、戴（たい）さんとか楊（よう）さんとか、いろいろな人に会（あ）いました
联欢会上，碰到了小戴、杨先生等许多人

语法

ば形

用于表示假定条件

  • 一类动词：把基本形的最后一个音（う段）变成相应的 え段上的音，再加ば

  • 二类动词：把基本形的 “る” 变成 “れば”

  • 三类动词：把 “来（く）る” 变成 “来（く）れば”， “する” 变成 “すれば”

  • 一类形容词：把词尾 “い” 变成 “ければ”

  • 二类形容词：在词尾加 “であれば”

类形 基本形 ば形
一类动词 書く かく かけば
急ぐ いそぐ いそげば
飛ぶ とぶ とべば
読む よむ よめば
死ぬ しぬ しねば
待つ まつ まてば
売る うる うれば
買う かう かえば
話す はなす はなせば
二类动词 食べる たべる たべれば
見る みる みれば
寝る ねる ねれば
三类动词 来る くる くれば
する する すれば
一类形容词 ない ない なければ
楽しい たのしい たのしければ
二类形容词 綺麗 きれい きれいであれば
親切 しんせつ しんせつであれば

小句1 ば，小句2

表示小句2成立必须以小句1的事态产生为条件

**原则上**小句2句尾不能够使用表示意志、希望、命令、请求的表达方式（たら可以）

  • 小句1的谓语为状态性谓语时可以使用。状态性谓语：いる、ある、できる、表达状态（部分自动词ている、〜ない、部分形容词）

  • 小句1与小句2的主语不同时可以使用

たら / と 后面的小句是过去形式，表示发现了某种情况，ば 没有这样的用法

窓（まど）を開（あ）けたら / 開けると、富士山（ふじさん）が見（み）えました

窓を開ければ、富士山が見えました　X


雨（あめ）が　降（ふ）らなけれ　ば、ハイキングに行（い）きましょう
要是不下雨的话，就去郊游吧（处于不下雨的状态）

雨が降れば、窓（まど）を閉（し）めてください　X
如果下雨，请关上窗户（下雨的动作）

この大会（たいかい）で　優勝すれば、オリンピックに出場することができます
如果在这次大会上获胜，就能够参加奥运会

李（り）さんに　聞（き）けば、分（わ）かりますよ
问小李就知道了

部屋（へや）がもう少（すこ）し　広（ひろ）ければ　いいのですが
房间要是再大一点就好了

如果日语考试超过90分，暑假就可以去旅游
日本語の　テスト/試験（しけん）　で、90点（てん）以上（いじょう）の成績（せいせき）取（と）れば、夏休みに旅行（りょこう）へ行けます

小句1（简体形）なら，小句2

用于以对方的某种意志、愿望、兴趣或嗜好为前提而提出某种建议，前项是兴趣或嗜好时，なら 和 たら 可以通用

小句2 为判断、命令、提议等表示说话者主观立场的内容

小句1 为二类形容词小句或名词小句时，把简体形的 “だ” 换成 “なら”（綺麗だ　ー　綺麗なら）

天安門へ行くなら、地下鉄が便利ですよ

今日（きょう）は忙（いそが）しいんですが…
今天很忙…
ーー　忙しいなら、行かなくてもいいですよ
—— 忙的话不去也行

今夜（こんや）、もし暇（ひま）なら、いっしょに食事（しょくじ）に行きませんか
今晚要是有空，一起去吃饭好吗？

会議（かいぎ）の資料（しりょう）はどこですか
会议资料在哪儿？
ーー　昨日（きのう）のは知（し）りませんが、今朝（けさ）の会議（かいぎ）の資料（しりょう）なら、机（つくえ）の上（うえ）に置（お）いておきました
—— 昨天的不知道，今天早上的会议资料，放在桌子上了

彼（かれ）が犯人（はんにん）でないなら、すぐ釈放（しゃくほう）してください
他如果不是犯人就马上放了

名词 でも　[示例]

でも 除了表示全面肯定和极端例子之外，还用于列举几个选项中有代表性的一项，一般不用于表示过去时态的句子（询问较多，偶尔自言自语，翻译成~之类）

映画でも見に行きませんか　/　映画でも見に行こうかな [自言自语]

運動（うんどう）でもしたらどうですか　/　運動でもしようかな [自言自语]
运动运动怎么样？

コーヒーでも飲（の）みませんか　/　コーヒーでも飲もうかな [自言自语]
喝点咖啡什么的怎么样？

私は昨日（きのう）喫茶店（きっさてん）でコーヒーでも飲みました　X

我要不要学会日语呢？
日本語でも勉強しようかな

名词1/小句1 とか 名词2/小句2 とか

用于列举同样性质的几个例子

パーティーで、戴さんとか楊さんとか、いろいろな人に会いました

コートとか靴（くつ）とか、たくさん買（か）いました
买了外套，鞋等好多东西

海外旅行（かいがいりょこう）に行（い）きたいとか、新（あたら）しい車（くるま）が欲（ほ）しいとか、ぜいたくなことばかり言（い）っています
想去海外旅游啦，想要新车啦，尽说些奢望

名词 とか 名词 とか 与 名词 や 名词 など 基本相同

とか 能连接名词和小句，や 只能连接名词

とか 只用于口语，や 用于口语和书面语

表达及词语讲解

買（か）っちゃった

〜ちゃった 是 〜てしまった 的口语形式。语义基本相同，但令人不愉快的语气更弱

2時間（じかん）ちょっとですね

表示 “两小时多一点儿”。ちょっと 常用于口语，可以接在时间词和数量词后

さすが

表示受到高度评价的人或物名副其实，确实如别人所评价的那样
