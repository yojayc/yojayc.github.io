## 语法

### 小句1 て/で 小句2 [原因、理由]

**て形** 除了用于[**连接多个动作**](https://yojayc.github.io/2026/01/26/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC14%E8%AF%BE/#%E5%8A%A8%E3%81%A6-%E5%8A%A8%EF%BC%88%E7%9B%B8%E7%BB%A7%E5%8F%91%E7%94%9F%EF%BC%89)和表示[**并列**](https://yojayc.github.io/2026/01/29/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC16%E8%AF%BE/#%E4%B8%80%E7%B1%BB%E5%BD%A2-%E3%81%A6-%E4%B8%80%E7%B1%BB%E5%BD%A2-%E5%B9%B6%E5%88%97)外，还可以用来表示**原因、理由**。

即 小句1 是 小句2 的原因、理由。**<span style="color:red">小句2不能是祈使句</span>**

> [**ので**](https://yojayc.github.io/2026/03/11/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC30%E8%AF%BE/#%E5%B0%8F%E5%8F%A51-%E3%81%AE%E3%81%A7%EF%BC%8C%E5%B0%8F%E5%8F%A52) **客观**地表达原因，[**から**](https://yojayc.github.io/2026/01/23/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC11%E8%AF%BE/#%E3%81%8B%E3%82%89-%E5%92%8C-%E3%81%A0%E3%81%8B%E3%82%89-%E8%AF%B4%E6%98%8E%E5%8E%9F%E5%9B%A0%E3%80%81%E7%90%86%E7%94%B1) **主观**地表达

遅くなって、すみません

最初（さいしょ）は言葉（ことば）が　通（つう）じ**なくて**、とても困（こま）りました　
开始的时候语言不通（言葉が通じない），非常困难

森（もり）さんは頭（あたま）が　痛（いた）くて、会社（かいしゃ）を休（やす）みました
森先生头疼，休息了

説明（せつめい）が　上手（じょうず）で、よく分（わ）かりました
解释得很好，我完全清除了

父（ちち）は大阪（おおさか）の　出身（しゅっしん）で、いつも関西弁（かんさいべん）で話（はな）します
我父亲生在大阪，所以总说关西话

他是美国人，所以总用英语说话
彼はアメリカ人で、いつも英語で話します/話しています

### 动词（基本形）の / 名词 に

**〜に/〜动词（基本形）+ の [形式名词] に** 可以用来表示**用途**和**基准**（比较的参照物）

表示**用途**时，前面是具体说明用途的名词，后面一般是 **使（つか）います** 等动词

この写真は何（なん）**に**使いますか
这张照片做什么用？
ーー　パスポートの申請に使います

この写真はパスポートを　申請するの　に使います

　　　　　　　　卵焼きをの料理に
このフライパンは卵焼（たまごや）きを　作（つく）るの　に使います
这个平底锅是煎鸡蛋用的

表示**基准**时，前面的名词是基准，后面一般是**评价的形容词**

スーパーが近（ちか）いので、このマンションは買（か）い物（もの）に便利（べんり）です
离超市很近，这个公寓出去购物非常方便

　　　　　　　　　買い物に
このマンションは　買い物するの　に便利です

この本（ほん）は、大人（おとな）には易（やさ）しいです。しかし、子供（こども）には難（むずか）しいです
这本书对大人来说很容易，对孩子来说太难了

这件衣服是参加派对时穿的
この服はパーティーに参加（さんか）するのに着（き）る服です

### 名词 <span style="color:red">ばかり</span> 动词 / 动词（て形）<span style="color:red">ばかり</span>います

**名词 + ばかり** 表示**所列举的事物<span style="color:red">全部相同</span>**

何（なん）で　野菜（やさい）ばかり　食（た）べているんですか
为什么光吃蔬菜？
ーー　今（いま）、ダイエット中（ちゅう）なんです
—— 正在减肥

林（はやし）さんはカラオケで　古（ふる）い歌（うた）ばかり　歌います
林先生唱卡拉OK尽唱些老歌

**动词て形 + ばかりいます 表示总是发生<span style="color:red">同样的事情</span>或总是进行<span style="color:red">同样的动作</span>**

張さんは毎日お酒を飲んでばかりいます

森（もり）さんはいつも失敗（しっぱい）してばかりいます
森先生老是出岔子

小林光吃了水果
小林（こばやし）君（くん）は果物ばかり食べています

### ばかり 和 [だけ](https://yojayc.github.io/2026/03/20/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC35%E8%AF%BE/#%E5%90%8D%E8%AF%8D-%E3%81%A0%E3%81%91)

#### 数量词 + ばかり / だけ

**数量词 + ばかり** 表示**左右**

三日（みっか）ばかり休みました
休息了三天左右

**数量词 + だけ** 表示**限定**

三（さん）つだけ食（た）べました
只吃了三个

#### 后接否定

**ばかり <span style="color:red">不能</span> + 否定**

**だけ + 否定**，表示 **只…没有…**

寿司（すし）だけ食（た）べませんでした
除了寿司都吃了

#### 前接て形

**て形 + ばかり** 表示**反复进行某事**

お茶（ちゃ）ばかり飲（の）んでいます
总是在喝茶

**て形 <span style="color:red">不能</span> + だけ**

#### ばかり 具有感情色彩

ばかり 具有**不太理想、抱怨**的感情色彩

### <span style="color:red">小句（动词简体形）の</span>が + 見（み）えます / 聞（き）こえます

**<span style="color:red">形式名词</span>が 見えます / 聞こえます** [自動詞]

見えます 聞こえます 的对象也可以用 **小句 + の** 表示

空港の入り口に警官が立っているのが見えます

隣（となり）の部屋（へや）で子供（こども）たちが騒（さわ）いでいるのが聞（き）こえますか
能听到隔壁房间孩子们吵囔的声音吗？
ーー　はい、よく聞こえます
—— 能，听得很清楚

彼（かれ）が車（くるま）に乗（の）ったのが見（み）えました
我看到他上了车

> 小句的谓语是一类形容词时用 **一类形容词 + の**
>
> 谓语是二类形容词时用 **二类形容词 + <span style="color:red">な</span> + の**

森（もり）さんが　うれしいの　が分（わ）かります
我知道森先生很高兴

日本語（にほんご）が　簡単（かんたん）なの　がよく分かりました
我明白了日语很简单

## 表达及词语讲解

### ほんと

本当（ほんとう）的口语缩写

### とにかく

表示虽然事情很多，但**不一一罗列，优先陈述某事**

### ぺらぺら

ぺらぺら 表示语言非常流利

### 量（はか）り売（う）り

日本以前蔬菜、佐料是按斤卖，现在一般都以盒装和袋装为单位销售
