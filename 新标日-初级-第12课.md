## 第12课小李比森先生年轻

### 课文

李さんは　森さんより　若（わか）いです
小李比森先生年轻

日本（にほん）より　中国（ちゅうごく）の　ほうが　広（ひろ）いです
和日本相比，中国更辽阔

神戸（こうべ）は　大阪（おおさか）ほど　にぎやがでは　ありません
神户没有大阪那么繁荣

スポーツの　中（なか）で　サッカーが　いちばん　おもしろいです
在各种体育活动中，足球最有意思

### 语法

#### Aは Bより 一类形/二类形です

A比B**更具有形容词所表示的性质**时使用


李さんは　森さん　より　若いです

中国は　日本　より　広いです
中国比日本辽阔

上海（シャンハイ）は　北京（ペキン）　より　にぎやがですか
上海比北京热闹吗？

<br>

程度差异较大时用副词“**ずっと**”强调

中国は　日本　より　ずっと　広いです
中国比日本辽阔得多

#### Aより、B<span style="color:red">の</span>ほうが　一类形/二类形です

与A相比，B更加的……


日本　より　中国　の　ほうが　広いです

北京　より　東京（とうきょう）　の　ほうが　暖（あたた）かいです
与北京相比，东京更暖和

大阪（おおさか）　より　東京　の　ほうが　にぎやかですか
与大阪相比，东京更热闹吗？

#### AはBほど　一类形く　ないです/二类形では　ありませんです

A没有B那么……


東京の　冬（ふゆ）は　北京の　冬　ほど　寒（さむ）くないです
东京的冬天没有北京的冬天那么冷

神戸は　大阪　ほど　にぎやか　では　ありません
神户没有大阪那么繁华

私の　部屋（へや）は　王さんの　部屋　ほど　きれい　<span style="color:red">**では　ありません**</span>
我的房间没有小王的房间那么整洁

#### Aの中（なか）で、Bが一番（いちばん）一类形/二类形です

在A的范围内，B是最……的


スポーツ　の　中で　サッカー　が　一番　おもしろいです

日本料理（にほんりょうり）　の　中で　寿司（すし）　が　一番　おいしいです
日式饭菜中，寿司最好吃

<br>

A表示**场所或时间**，后面不加“の中”

使用 **A[场所/时间]で 一番（いちばん）一类形/二类形 Bは　Cです** 的句型


日本で　一番　高（たか）い　山（やま）は　富士山（ふじさん）です
日本最高的山是富士山

一年（いちねん）で　一番　寒（さむ）い　月（つき）は　２月（にがつ）です
一年当中最冷的月份是2月份

> 日语的“一番”只能用于 “〜の 中で” “〜で”等**明确指出比较对象或范围**的情况

#### Aと　Bと　どちらが　一类形/二类形ですか

询问**两种**事物中哪种更具有该句形容词的性质时使用

> どちら　是二选一时使用的疑问句，使用“**〜の　ほうが〜**”回答
>
> 两者**程度相当**，使用“どちらも　〜です”回答
>
> 口语中有时将“どちら”说成“どっち”


日本語　と　英語（えいご）　と　<span style="color:red">どちら</span>が　難（むずか）しいですか　日语和英语哪个更难学？
ーー日本語　の　ほうが　難しいです		日语更难学

コーヒー　と　紅茶（こうちゃ）　と　<span style="color:red">どっち</span>が　好（す）きですか	咖啡与红茶更喜欢哪个？
ーー紅茶　の　ほうが　好きです	更喜欢红茶
ーーどちらも　好きです	哪个都喜欢

> 询问范围为**两种**事物，无论人、物还是时间，都可以用“どちらが”
>
> 询问**三种以上**的事物时不能使用“どちら”

#### どの　名/いつ/どれ/だれ/何（なに）が　一番（いちばん）　一类形/二类形ですか

询问**三种以上**事物中哪一个更具有该句形容词的性质

- 询问人用“だれが”
- 询问物用“どの ~が”　“どれが” 或 “何が”　
- 询问时间用“いつが”


この　中（なか）で　<span style="color:red">どの　料理（りょうり）　が</span>　いちばん　おいしいですか　这些菜中，哪个菜最好吃
ーーこの　料理　が　いちばん　おいしいです　这个菜最好吃

季節（きせつ）の　中で　<span style="color:red">いつ　が</span>　いちばん　好きですか　四个季节中，你最喜欢哪个？
ーー私は　春（はる）が　いちばん　好きです　我最喜欢春天

**被选择的事物在上下文中提到或在眼前，使用“どの ~が”或“どれが”，否则使用“何（なに）が”**

リンゴと　ナシと　バナナの　中で　どれが　いちばん　好きですか
苹果、梨和香蕉当中你最喜欢什么？

果物（くだもの）の　中で　何が　いちばん　好きですか
水果当中你最喜欢什么？

### 表达及词语讲解

#### 最近（さいきん）

日语的“最近”表示过去到现在为止，不表示将来

#### 〜が　いいです

〜が　いいです　表示“～比较好”，“我喜欢～”
