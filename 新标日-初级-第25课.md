## 课文

これは<span style="color:red">明日（あした）会議（かいぎ）で使（つか）う</span>資料（しりょう）です
这是明天会议要用的资料

<span style="color:red">私が明日乗（の）る飛行機（ひこうき）</span>は中国航空（ちゅうごくこうくう）です
我明天乘坐的飞机是中国航空公司的

<span style="color:red">中国で買（か）ったCD（シーディー）</span>を友達（ともだち）に貸（か）しました
我把在中国买的CD借给朋友了

<span style="color:red">操作（そうさ）が簡単（かんたん）な</span>パソコンが欲（ほ）しいです
我想要操作简单的个人电脑

## 语法

### 动（<span style="color:red">简体形</span>）+名词/小句（<span style="color:red">动词简体形</span>）+名词

动词或由动词做谓语的小句（动词小句）**修饰名词**时，用动词的![**简体形**](https://yojayc.github.io/2026/02/09/%E6%96%B0%E6%A0%87%E6%97%A5-%E5%88%9D%E7%BA%A7-%E7%AC%AC22%E8%AF%BE/#%E6%95%AC%E4%BD%93%E5%BD%A2%E5%92%8C%E7%AE%80%E4%BD%93%E5%BD%A2)

表示**某种习惯或将要发生的动作**时用动词的**基本形**，表示动作**已经完成**时用动词的“**た形**”

これは　明日会議で使う　資料　です

あれは　森（もり）さんが今夜（こんや）泊（と）まる　ホテル　です
那是今晚森先生要入住的宾馆

あれは　倒産（とうさん）した　会社（かいしゃ）　です
那是一家倒闭的公司

阿飞老师的头像是甩子画的画
トビ先生のアイコンは SZ描（か）いた絵です

**名词前不用敬体形**

这个座位是森先生刚才坐过的地方
さっき森さんが<u>座（すわ）った</u>場所（ばしょ）
さっき森さんが<u>座りました</u>場所　X

这是森先生不知道的新闻
森さんが<u>知（し）らない</u>ニュース
森さんが<u>知りません</u>ニュース X

**名词前不能加“の”**

森先生要坐的地方
森さんが座る場所
森さんが座る<u>の</u>場所 X

那是昨天晚上住的宾馆
あれはゆうべ泊（と）まったホテルです
あれはゆうべ泊まった<u>の</u>ホテルです X

### 小句（<span style="color:red">动词简体形</span>）+名词　は　名词/形容词です

动词小句修饰名词形成的**名词短语**常常用来做**主语**

王（おう）さんは中国人（ちゅうごくじん）です
<span style="color:red">私が明日会（あ）う</span>王さんは中国人です

この会社（かいしゃ）は貿易会社（ぼうえきがいしゃ）です
<span style="color:red">李（り）さんが投資（とうし）した</span>この会社は貿易会社です

<span style="color:red">私が明日乗る</span> 飛行機は　中国航空　です

<span style="color:red">小野（おの）さんが作（つく）る</span> 料理（りょうり）は　おいしいです
小野女士做的菜很好吃

<span style="color:red">森さんが買（か）った</span> 本（ほん）は　これです
森先生买的书是这一本

> 日语名词前常常有**较长的定语**

昨日（きのう）父（ちち）はデパートで
